# Xiaomi-MiMoAi-discord

複数のAPIを使ってるので、APIを取得してください。

1.Xiaomi-MiMoAI-API
2.Tavily-API

この二つとdiscord-botのAPIを取得し、python上部にあるAPIに貼り付けして
サーバーやホスト環境から起動してください。

Since multiple APIs are used, please obtain the APIs.
